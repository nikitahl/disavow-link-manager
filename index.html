How to use:

  1. Export and upload CSV file containing external links from Google Search Console;
  2. In the generated table select those links you with to disavow;
  3. (Optional) Upload existing disavow file;
  4. Copy selected links to clipboard or download ready to use file;
  5. Upload the file with links to Google Search Console's disavow links tool.
